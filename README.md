# 🎓 Online Exam Evaluation System

## 📌 Project Overview
The Online Exam Evaluation System is a Java-based desktop application designed to manage student exam operations efficiently. It helps users add students, assign marks, evaluate results, and manage exam records in a structured way.

This project is built using Core Java (Swing/AWT) and MySQL database for backend data storage.

---

## ✨ Features
- Add new student details
- Assign and manage exam marks
- View student results
- Store data using MySQL database
- Simple and user-friendly GUI (Swing)
- Basic data management system

---

## 🛠️ Technologies Used
- Java (JDK 8+)
- Java Swing / AWT (GUI)
- MySQL Database
- JDBC (Java Database Connectivity)
- Git & GitHub
